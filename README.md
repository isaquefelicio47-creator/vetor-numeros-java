# Vetor de Números em Java
## Descrição
Programa que lê 5 números inteiros e realiza operações como:
- Maior número par
- Menor número ímpar
- Soma e média
## Como executar
1. Compile o programa
2. Execute a classe principal
## Exemplo de uso
Entrada:
5, 8, 3, 10, 7
Saída:
Maior par: 10
Menor ímpar: 3
Soma: 33
Média: 6.6
